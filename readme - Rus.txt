Serv - простой сервер на Linux, который ожидает клиента, пересылает большой файл, отключается от клиента и ждёт следующего.
Cli - простой клиент на Windows, который подключается к серверу, принимает файл и выводит его размер.
CliAsyn - асинхронный клиент на Windows, который подключается к серверу, принимает файл и выводит его размер.

Проблема возникает в связке CliAsyn(windows) - Serv(linux).
Часть данных теряется, но сервер этих потерь не видит.
Если перевести и сервер на Windows, то потерь не будет.
Так-же потерь не будет, если использовать синхронный клиент(Cli).
